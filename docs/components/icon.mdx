## Options

### Size

The `size` can be 16px, 24px, or 32px. The size sets the width and height of the SVG.

```jsx
// preview
import { Icon, theme } from "@washingtonpost/wpds-ui-kit";
import Asset from "@washingtonpost/wpds-assets/asset/chevron-right";

export default function Example() {
  return (
    <>
      <Icon label="" size="16">
        <Asset fill={theme.colors.primary} />
      </Icon>
      <Icon label="" size="24">
        <Asset fill={theme.colors.primary} />
      </Icon>
      <Icon label="" size="32">
        <Asset fill={theme.colors.primary} />
      </Icon>
    </>
  );
}
```

### Label

The `label` is required for screen readers. It is used to describe the icon to screen readers. It is also used as the `aria-label` for the icon. It is recommended to use a short, descriptive label. For example, "Go to Next Page".

```jsx
// preview
import { Icon, theme } from "@washingtonpost/wpds-ui-kit";
import Asset from "@washingtonpost/wpds-assets/asset/chevron-right";

export default function Example() {
  return (
    <>
      <Icon label="Go to Next Page" size="16">
        <Asset fill={theme.colors.primary} />
      </Icon>
      <Icon label="Go to Next Page" size="24">
        <Asset fill={theme.colors.primary} />
      </Icon>
      <Icon label="Go to Next Page" size="32">
        <Asset fill={theme.colors.primary} />
      </Icon>
    </>
  );
}
```
