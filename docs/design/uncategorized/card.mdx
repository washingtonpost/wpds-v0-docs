# Card

<!-- Description of component -->
<p className="font-xs font-light font--subhead">
Cards are surfaces that display content and actions on a single topic. They should be easy to scan for relevant and actionable information. Elements, like text and images, should be placed on them in a way that clearly indicates hierarchy.
</p>

## Usage guidelines

<div className="mt-lg mb-sm">

### Padding
When using the card component text content should have padding and the image should 
be full bleed to the container. 

</div>
<div className="flex w-100 mb-0">
  <Guide className="w-100 b br bc-gray-light" guidance="don't">
    <Container className="flex-column pa-sm">
      <div>
        <Card style={{width:200}} className="bg-white pa-xs overflow-hidden">
          <img src="https://picsum.photos/200/150"/>
          <h6 className="pb-0 font--headline font-md">Card Title</h6>
          <p className="pt-0">Simple paragraph about this picture.</p>
        </Card>
      </div>
    </Container>
  </Guide>
  <Guide className="w-100" guidance="do">
    <Container className="flex-column pa-sm">
      <div>
        <Card style={{width:200}} className="bg-white overflow-hidden">
          <img src="https://picsum.photos/200/150"/>
          <h6 className="pb-0 pl-xs pr-xs font--headline font-md">Card Title</h6>
          <p className="pb-sm pl-xs pr-xs">Simple paragraph about this picture.</p>
        </Card>
      </div>
    </Container>
  </Guide>
</div>
