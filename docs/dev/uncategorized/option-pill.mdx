# Option Pill

<!-- Description of component -->
<p className="font-xs font-light font--subhead">
An Option Pill can represent an input, attribute, or action. An Option Pill 
allows users to trigger actions, like links to points of interest. While 
buttons are expected to appear consistently and with familiar calls to action, 
Option Pills can appear dynamically, either individually or as a group.

## How to use

```jsx
import {
  OptionPill,
  OptionPills
} from "@washingtonpost/site-components/core/option-pill";

<OptionPills as="div">
  <OptionPill href="#politics">Politics</OptionPill>
  <OptionPill href="#technology">Technology</OptionPill>
  <OptionPill href="#business">Business</OptionPill>
</OptionPills>;
```

## Wrapping

Itâ€™s common to create a list of Pills as anchors within an unordered list. To do this, you'll need to wrap the OptionPill inside of a `li` tag. The `OptionPills` component allows you to pass any children and those nodes will receive the `mr-xs` class to maintain the accurate spacing for a group of Option Pills outlined by the design system.

```jsx
<OptionPills as="ul" className="pl-0 ma-0">
  <li>
    <OptionPill href="#boop1">List link #1</OptionPill>
  </li>
  <li>
    <OptionPill href="#boop2">List link #2</OptionPill>
  </li>
</OptionPills>
```

## Adding props to an OptionPill and to OptionPills

The parent `OptionPills` and the child `OptionPill` both accept conditional props in case you need to add additional attributes for styles, aria labels, testing, etc. This can be useful when iterating over your data to produce a set of `OptionPill` components.

```jsx
export const MyCustomOptionPill = () => {
  const items = [
    {
      href: "#politics",
      content: "Politics",
      backgroundColorClass: "bg-red"
    },
    {
      href: "technology",
      content: "Technology",
      backgroundColorClass: "bg-green"
    },
    {
      href: "#business",
      content: "Business"
    }
  ];

  return (
    <OptionPills
      aria-label="test-aria"
      as="div"
      className="flex w-100 overflow-auto pl-0"
      data-cy="option-pills"
      data-qa="option-pills"
      style={{ flexWrap: "nowrap" }}
    >
      {items.map(({ href, content, bg }) => (
        <OptionPill
          aria-label={`Select ${content}`}
          className={backgroundColorClass}
          data-cy={`option-pill-${content}`}
          data-qa={`option-pill-${content}`}
          href={href}
          key={href}
        >
          {content}
        </OptionPill>
      ))}
    </OptionPills>
  );
};
```
